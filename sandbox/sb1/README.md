## Pourquoi utiliser React ?
Car une application de base n'est pas très maintenable, le plus gros soucis est la gestion des
dépendances.

React est né ppur simplifier la création d'IHM complexes et pour gérer
les mises à jour de l'IHM de manière efficace

## Création d'un projet React

```
npm create vite@latest
```

est la commande à faire dans le dossier dans lequel on souhaite créer le projet.

## Composants React

Un composant doit toujours démarrer par une majuscule sinon React ne sait pas
faire la différence entre du HTML de base et nos composants

Un composant react est généralement une **fonction** qui doit être **exportée** à la fin du fichier
